앱에서 제한된 미디어 라이브러리 접근 권한만 부여하셨나요? 대단한 일은 아니지만, 설정을 변경하고 접근 권한을 추가하려는 경우에 대비하여 미디어 선택기에 프롬프트를 추가했습니다. 판독하기 쉽게 Dynamic Type으로도 프롬프트가 제공됩니다.

활동 로그 달력과 글 예약 달력에도 VoiceOver와 Dynamic Type을 더 정확하게 표시했습니다. 이를 통해 시각적 접근성 향상이 필요한 사람들의 사용자 환경이 개선됩니다.

블록 편집기에도 일부 유용한 업데이트를 적용했습니다.

- 버튼 블록에서 버튼이 왼쪽, 오른쪽 또는 가운데에 맞춰지지 않았던 문제를 해결했습니다.
- 커버 블록의 색상 설정 번역을 추가했습니다.
- 이미지를 선택하면 더는 비디오 썸네일이 표시되지 않습니다(반대의 경우도 마찬가지).
- 지원되지 않는 블록 편집기 오른쪽 하단의 도움말 버튼을 숨겨 혼란을 방지했습니다.
- 텍스트 블록에서 텍스트와 배경 색상을 여러 가지 사용할 때 색상 조합을 일부 사람들이 판독하기 어려울 수도 있는 경우 알림이 표시됩니다.
- 선택하지 않은 텍스트의 색상을 변경할 때 발생하던 소수의 예외적인 문제를 해결했습니다.

몇 가지 소소한 문제와 충돌도 해결했습니다.

- 이제는 최신 소식을 더 쉽게 파악할 수 있도록 주간 총정리 알림이 표시됩니다.
- 이제는 페이스북을 배포 기능에 연결하면 로그인됩니다.
- 이제는 파일 이름에 저작권 기호가 있는 이미지가 적절히 업로드되거나 공유 시트에 표시됩니다.
